# esusatyo dotfiles

## Usage

```zsh
$ /bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"
$ git clone https://github.com/esusatyo/dotfiles.git ~/dotfiles
$ cd ~/dotfiles && ./install.sh
```
